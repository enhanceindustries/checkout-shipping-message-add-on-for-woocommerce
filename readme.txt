== Description ==
This add-on will allow you to add a custom message to WooCommerce under that shipping totals shipping section of your checkout.

The message will be wrapped by id shipping_message and class woocommerce-shipping-shipping-message for your theme to style

Proudly supported by [Enhance Industries](http://enhanceindustries.com/)

= Features =

Allows global custom message on the shipping totals area when checking out of Woocommerce
